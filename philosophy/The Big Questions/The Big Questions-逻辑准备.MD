# 大问题（The Big Questions）：简明哲学导论 第六版

# 逻辑准备

逻辑的主要问题是对**论证**进行分类，即把坏的论证归为一类，把好的论证归为另一类。

	by 查尔斯·桑德斯·皮尔士
	
哲学是由观念、重要理论和洞察力所组成的，它们必须在演讲或写作中以尽可能清晰的方式表达出来。**好的哲学表述的关键是论证**。

日常语言中，我们有时会觉得某种论证充满了敌意与怨恨，这未必是事实。一则好的论证完全能够以一种冷静、平和的方式表达出来。**最好的论证**总是由一种被我们称为**逻辑**的缜密思维过程来规定的，通常把逻辑定义为“**进行正当推理的科学（和过程）**”。

**哲学家会运用各种不同的论证和论证类型**。当然，政客、商人、脱口秀电视节目主持人以及我们每个人都会这样做。有些论证是通过**例子**（“举个例子”），和**类比**（“生活就像一部小说”）进行的；有些论证基于广泛的**科学研究**；有些论证非常**抽象**，在很大程度上还只是**言辞**上的；有些论证仅仅是基于对对手的**恶意攻击**；还有一些则徒有其表，谈不上论证。

一般来说，我们可以根据不同表述之间的关系把论证分为两种——好的论证和坏的论证。（真与假使用陈述本身而非论证）

哲学家和逻辑学家把论证分为两类：**演绎**和**归纳**。

## 演绎

**在演绎论证中，结论的真是由其他几条陈述的真来保证的**。如中学几何中的定理证明。有些陈述是一开始就假定为真的（如公理），于是新的陈述就经由几条已经建立起来的**推理规则**——亦即思维规律——被演绎或推导出来了。因此演绎论证是从一条真陈述推演出另外一条真陈述。演绎论证有时这样定义：**一条最终陈述（结论）的真完全由初始陈述（前提）的真来保证**。

PS：**命题**是指一个判断（陈述）的语义（实际表达的概念）。

最著名的演绎论证是传说中的**三段论**。如，

```
所有哲学家都是智慧的。
苏格拉底是哲学家。
---
因此，苏格拉底是智慧的。
```

其论证的形式是：

```
所有的P都是Q。
S是P。
---
因此，S是Q。
```

当一则演绎论证严格按照这种形式进行时，我们说它是**演绎有效的**或**有效的**，否则就是**无效的**。演绎保证我们的**结论至少是与前提同样可靠的**。对有些论证来说，即使前提和结论都是假的，该论证也可能是有效的。如

```
所有的牛都是紫色的。
苏格拉底是牛。
---
因此，苏格拉底是紫色的。
```

该论证有效，尽管其前提和结论都为假。**好的演绎论证必须具备为大家所公认的真的前提**。需注意，哲学家把**”有效“和”无效“严格用于评判论证**的正确与否上，而**真假**则适用于论证中的各种**陈述**。

如果一则论证既是有效的，前提又是真的，那么该论证就被称为**合理的**（好的论证）。如果一则论证的前提为假，或其本身是无效的，那么该论证被称为**不合理的**（坏的演绎论证）。因此，一则好的演绎论证需要：

1. 它是有效论证
2. 它的前提为真

## 归纳

归纳论证最常见的形式是由若干个别事例出发进行**概括**。

对于任何一则归纳论证来说，细心挑选个别事例，确保它们尽可能地不同（也就是说，按照社会学家们通常所说的“随机抽样调查”来进行）是很重要的。

根据证据**对结论的支持度**、所举**事例的质量**以及**概括的可信度**的不同，归纳论证可分为**强的**和**弱的**两种。**归纳论证不会被说成是有效或无效**，事实上，既然有效论证的定义是只要前提为真，结论就必然为真，那么归纳论证就不可能是演绎有效的（它虽不是演绎有效的，却有其他功能）。

虽然概括是最常见之方式，但归纳论证可用来支持任何事实陈述。比如你认为凯撒是公元前44年3月15日在罗马某人谋杀的，那么此陈述的论证就必须是归纳的。在一场审讯中，两种互相抵触的事实陈述（被告有罪和被告无罪）可能都存在证据支持。法官和评审团的归纳问题是，定罪的证据是否要比反对定罪的证据更为一致（即更强？）。值得注意的是，福尔摩斯的"惊人的演绎能力“，其实是一种从零碎的蛛丝马迹中获取事实论断的归纳推断能力。

* Induce: Specific -> General
* Deduce: General -> Specific

归纳论证中最重要的部分之一是**假说**，即一个实验应当予以证实或否证的陈述。我们已熟悉科学中的假说，但**事实上我们终生都在使用假说**。

* 早上寻找钥匙（“我确实把它放在书桌上了”）
* 对黑洞的存在与性质的思考

有上述讨论可知，大多数所谓的论证其实根本不是论证。明确表述是论证的重要前提，但不等同于论证。**论证超出了意见本身**，它通过从其他陈述或经验证据进行演绎来为该意见提供支持。**大多数归纳论证都需要许多例子**，同时还要应对那些并不符合假说的例子。

类似地，许多人似乎认为**诉诸权威**就可以万事大吉了，但在**大多数哲学争论中，受到质疑的却正是权威**。如果某人坚持说，上帝之所以存在，是因为《圣经》是这样告诉我们的，那么问题就立刻指向了《圣经》的权威（况且一个不信上帝的人可能也不会接受《圣经》的权威）。如果某个人之所以会为某种政治立场辩护，是他的母亲曾经这样说过，或是托马斯·杰斐逊曾经这样说过，那么问题立刻就指向了说话者的母亲或杰斐逊的权威。**哲学的一个主要功用就是让我们对权威进行质疑，亲自去思考我们应当相信什么，不应当相信什么**。单单是诉诸权威并不必然表示对于该权威的尊重，而是表示**对我们自己的不尊重**。真理总是能够依凭自身站住脚的，如果**我们不愿对自己的想法进行批判性地检验，那么它们是没有多大价值的**。（未经省察的人生没有价值？）

## 对论证进行批判

哲学中非常关键的活动就是**批判**。批判并不必然意味着对某人某事持反对意见；批判意味着对某条陈述进行认真地查验，看看对它的论证是否的确是好的论证。

对于归纳概括，**反例**是一种特别有力的方式。即使所有论证似乎都是合理的，某种哲学也可能会由于不一致而出问题，即从不同论证出发会得出相互**矛盾**的结论。如”上帝能否造出一座大得连他也搬不动的山？“，这是一种**归谬法**形式的论证（对于”上帝是全能的“）。

然而，即使一种哲学并不包含直接的不一致或悖论，它也可能是不连贯的，即各个陈述之间并没有什么关系，或陈述本身并没有什么意义。有的哲学可能会被指责为”**循环论证**“，还有“同义反复”。有的论证可能会具有**人身攻击**色彩，它们意在人而非问题。

在强调批判时，也应注意哲学是一项特别需要合作的事业。如果一个朋友从未批评过你，从不与你争论，那么我们是不会信任他的。每个人的意见都应受到尊重，但这并不意味着每个人的意见都具有同等的价值（思考之深度与论证之质量）。**对于合作的尊重、就论证的重要性达成共识以及诚恳地提出否定意见是我们所向往的生活的前提**。

#### 悖论

**悖论是指基于显然是好的论证而得出自相矛盾的或荒谬的结论**。

* 这句话是假的。
* 理发师悖论。
* 上帝全能与山
* 阿基里斯与龟

#### 逻辑的基本概念

**论证**是一系列用以支持某种现点或思想的断言或陈述。所谓**结论**，是指为所有其他陈述所支持的断言。于是那些陈述就成为接受该结论的**理由**，那些（为了论证）被假定为真的陈述被称为**前提**。论证既可以是**演绎**的，也可以是**归纳**的：如果论证既是有效的，其前提又为真，那么演绎论证就能保证结论为真；而**归纳论证不能保证其结论的真实性**，无论证据看上去是多么可靠。演绎论证可以被称为**有效的**或**无效的**（论证绝没有真假之分），无效论证也被称为**谬误**。个别断言或陈述有**真**或**假**之分（它们不能被说成是有效或无效）。如果一则演绎论证既是有效的，其前提又为真，那么该论证就是**合理的**（否则就是**不合理的**）。得到了证据很好支持的归纳论证被称为**强的**，否则就被称为**弱的**。逻辑是对好的论证的运用与研究。

**同义反复**是一种平凡的真陈述。如：

* 如果一个人是自由的，那么他就是自由的。
* “玫瑰就是玫瑰。” （格特鲁德·斯泰因）
